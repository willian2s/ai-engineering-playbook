# 🛠 AI TOOLING GUIDE

Guia objetivo de uso das ferramentas de IA.

---

## 🟦 COPILOT (DEFAULT)

### Uso obrigatório inicial

Use para:

- desenvolvimento diário
- pequenas features
- refactors locais
- ajustes rápidos
- escrita incremental de código

---

### NÃO usar para

- arquitetura
- debugging complexo
- refactors multi-arquivo
- investigação profunda

---

## 🟢 CODEX (ESCALADO)

### Uso somente quando houver gatilho

Use para:

- debugging complexo
- refactors multi-arquivo
- mudanças arquiteturais
- integração entre sistemas
- problemas não resolvidos no Copilot

---

## 🧠 REGRA SIMPLES

- Copilot = sempre primeiro
- Codex = somente com gatilho válido
