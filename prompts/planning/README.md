# 🧠 Planning Prompts

Esta pasta contém os prompts responsáveis por transformar problemas em execução técnica estruturada.

Eles formam um **pipeline completo de engenharia assistida por IA**, focado em clareza, segurança e execução em produção.

---

## 🎯 Objetivo

Padronizar como problemas são:

1. Entendidos (sem suposições)
2. Validados (sem riscos ocultos)
3. Transformados em planos executáveis (sem overengineering)

---

## 🧠 Princípio central

> **Não executar antes de entender.
> Não escalar antes de validar.
> Não complicar sem necessidade.**

---

## ⚙️ Fluxo oficial

```bash
problem → breakdown → validation → execution → implementação
```
